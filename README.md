**AI Resume Analyzer:**

Description:
This is an AI powered web app that analyzes resumes and provides reallistic instant feedback, 
scoring, skill gap analysis, and job role suggestions using LLaMA 3.3 via Groq API.

Live Demo:
Go to this link and check it you will see User Interface. 
https://ai-resume-analyzer-6qzvozsdzphdcsehfpq6ec.streamlit.app

Features:
- Smart Scoring — Resume score out of 10
- Skill Gap Analysis — Discover missing skills
- Job Matching — Find best fitting job roles
- Improvement Suggestions — Actionable feedback
- Fast Analysis — Powered by LLaMA 3.3
  
Tech Stack:
Python | Streamlit | Groq API | LLaMA 3.3 | pdfplumber
